# Examples for Product Description Writer

This directory contains example scripts demonstrating how to use this project.

## Quick Demo

```bash
python examples/demo.py
```

## What the Demo Shows

- **`load_config()`** — Call load_config
- **`get_platform_configs()`** — Call get_platform_configs
- **`get_feature_benefit_map()`** — Call get_feature_benefit_map
- **`map_features_to_benefits()`** — Map product features to customer benefits.
- **`calculate_seo_score()`** — Calculate an SEO score for the given text.

## Prerequisites

- Python 3.10+
- Ollama running with Gemma 4 model
- Project dependencies installed (`pip install -e .`)

## Running

From the project root directory:

```bash
# Install the project in development mode
pip install -e .

# Run the demo
python examples/demo.py
```
